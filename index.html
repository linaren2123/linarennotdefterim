Ayarlar & Yerel Veri Yönetimi

Kütüphane Yedekleme ve Kurtarma

Verileriniz otomatik olarak tarayıcınızın IndexedDB ve LocalStorage alanlarında çift katmanlı saklanır. Ekstra güvence için verilerinizi bilgisayarınıza yedek dosyası (JSON) olarak kaydedebilir veya geri yükleyebilirsiniz.

Çalışma Alanını Sıfırla

Tüm yerel veritabanını temizleyerek varsayılan sade Türkçe başlangıç sayfalarını yükleyebilirsiniz. Bu işlem geri alınamaz.

GitHub Gist Senkronizasyonu

Notlarınızı ve klasörlerinizi GitHub Gist üzerinden yedekleyebilir ve cihazlar arası eşitleyebilirsiniz.

Token almak için GitHub hesabınızdan 'gist' yetkili bir Personal Access Token (classic) üretin.

Google Drive Yedeklemesi

Belgelerinizi Google Drive'da otomatik oluşturulan 'LinareN_Backups' klasörü altına zaman damgalı JSON dosyaları olarak yedekleyin.

Güvenlik için Google Cloud Console üzerinden API Anahtarına 'Google Drive API' kısıtlaması eklemeniz önerilir.
Bağlı Değil

WebDAV Bulut Yedeklemesi

Kişisel bulut sunucunuza (Nextcloud, ownCloud vb.) WebDAV protokolü üzerinden yedekleme yapın.

Uç nokta URL'si doğrudan yedekleme klasörünü hedeflemelidir (sonunda / olmalıdır).

Zamanlanmış Otomatik Yedekleme

Belgelerinizin Google Drive veya WebDAV üzerine düzenli aralıklarla yedeklenmesini planlayın (Bu işlem sadece uygulama tarayıcıda açıkken tetiklenir).

E-posta ile Not Gönderim Ayarları

Notlarınızı hızlıca e-posta olarak göndermek için alıcı adresini ve gönderim yöntemini yapılandırın. EmailJS entegrasyonu ile notlar doğrudan arka planda gönderilir.

Notlarınızın varsayılan olarak gönderileceği e-posta adresi.
EmailJS şablonunuzda 'to_email', 'note_title' ve 'note_content' değişkenleri kullanılmalıdır.
